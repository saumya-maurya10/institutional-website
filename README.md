Riverdale Institute of Technology- Institutional website
The Riverdale Institute of Technology website is a simple and user-friendly web project 
designed to provide essential information about the institution in an organized and 
interactive way. The website is developed using basic web technologies such as HTML, CSS 
and JavaScript. 

Features-
The project includes multiple sections such as Home , About, Courses, College Life, Entrance 
Exam, Registration and Contact. Instead of loading different web pages, the website uses a 
single-page design where content is dynamically displayed using JavaScript, making 
navigation fast and smooth. 

Overall, this project demonstrates the use of front-end web development techniques to 
create an informative, visually appealing, and interactive institutional website suitable for 
academic purposes.

Proposed design/ Architecture- 
The institutional website follows a simple front-end architecture based on a single page 
application concept. 

Structure- 
1) HTML(Contains all sections): 
Home
About 
Courses 
College Life 
Entrance Exam 
Registration 
Contact 
Each section is created using <section> tag 
 
3) CSS: 
Used for colours 
Layout 
Fonts 
Images 
Makes the website look attractive and user-friendly

4) JavaScript: 
Handles page switching 
Form response 
Makes the website interactive
